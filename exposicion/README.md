# Exposición de Cinemática

Este directorio contiene el material y el simulador para preparar la exposición de Cinemática.

## Abrir simulador

Cuando GitHub Pages esté activado en este repositorio, el simulador se podrá abrir desde este enlace:

[Abrir simulador en GitHub Pages](https://chinuxs.github.io/fisica-utn-tiro-oblicuo/)

También puedes ejecutar el simulador localmente abriendo:

- `exposicion/simulador/index.html`

## Qué incluye

- `simulador/`: aplicación web simple de tiro oblicuo.
  - `index.html`
  - `styles.css`
  - `script.js`
- `docs/`: documentación de apoyo para la exposición.
  - `guion_exposicion.md`
  - `resumen_para_estudiar.md`
  - `preguntas_profesor.md`

## Cómo usarlo

Abrir `exposicion/simulador/index.html` en un navegador para ejecutar el simulador.

Leer `exposicion/docs/README.md` para acceder al material de exposición y estudiar el guion.
