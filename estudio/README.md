# Material de estudio para Física I

Esta carpeta agrupa recursos pensados para preparar parciales y repasos de
Física I.

## Estructura

- `parciales/`: espacio para exámenes, simulacros y guías de estudio.
- `apuntes/`: apuntes y materiales teóricos.
- `ejercicios/`: ejercicios prácticos y problemas resueltos.

## Cómo usarlo

Guardar aquí los recursos que ayuden a estudiar las materias del curso sin
confundirlos con el simulador de la exposición.

## Nota

Esta área está diseñada para que compañeros encuentren rápidamente el material
de estudio, sin necesidad de revisar la implementación del simulador.
